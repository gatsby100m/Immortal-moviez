# Immortal Moviez 🎬

Immortal Moviez is a full-stack movie streaming UI with authentication and favorite movie storage using MongoDB, JWT, React, and Express.

---

## 🔧 Technologies Used

### Frontend:
- React
- Axios
- CSS

### Backend:
- Node.js
- Express.js
- MongoDB (via Mongoose)
- JWT (Authentication)
- Bcrypt.js (Password Hashing)
